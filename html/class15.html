  • RGB 주요 브라우저에 모두 지원. rgb(적색, 녹색, 청색)의 형태로 0~255의 정수로 된 퍼센트 값(0%~255%)을 가지게 된다.
  • RGBA IE9+, Firefox 3+, Opera 10+, Chrome, Safari에서 지원한다. rgba(적색, 녹색, 청색, 투명도)의 형태로, 0.0(투명)~1.0(불투명) 사이의 값.
  • HSL IE9+, Opera 10+, Firefox, Chrome, Safari에서 지원한다. HSL 색상은 hsl(색조, 채도, 명도)의 형태로 색조, 채도 및 밝기를 원통형의 좌표를 이용해 나타낸다. 색조는 색상환에서 0~360까지 - 0 혹은 360은 적색, 120은 녹색, 240은 청색을 나타낸다. 채도는 % 값을 가지며, 0%는 회색을 가지게 된다. 명도 또한 % 값을 가지며, 0%는 검정, 100%는 하얀색이 된다.
  • HSLA IE9+, Firefox 3+, Opera 10+, Chrome, Safari에서 지원한다. hsla(색조, 채도, 명도, 투명도)의 형태로 0.0(투명)~1.0(불투명) 사이의 값을 가지게 된다.
  • opacity CSS3에 추가된 속성으로 요소의 투명화 정도를 조절하는 속성이다.
