13일차 정리

  • border-style
  • 요소의 크기 지정 및 여백 처리하기
  • 다양한 속성으로 테두리 표시하기
  • 인용부호, 특정 문자열과 같은 콘텐츠 생성하기
  • list-style-image 속성 list-style-image 속성을 이용하면 마커(marker)로 자신만의 이미지를 사용할 수 있습니다. 예제 style .imageMarker { list-style-image: url("/examples/images/img_list_marker.png"); } /style
  • list-style-position 속성 list-style-position 속성을 이용하면 리스트 요소의 위치를 설정할 수 있습니다. list-style-position 속성의 기본 속성값은 outside로 설정되어 있습니다. 예제 style .outside { list-style-position: outside; } .inside { list-style-position: inside; } /style
  • list-style-type 속성 리스트 요소의 앞에 위치하는 숫자나 기호를 마커(marker)라고 합니다. list-style-type 속성을 이용하면 리스트에 다양한 마커(marker)를 적용할 수 있습니다. 예제 style .circle { list-style-type: circle; } .square { list-style-type: square; } .upperAlpha { list-style-type: upper-alpha; } .lowerRoman { list-style-type: lower-roman; } /style
  • direction: ltr : 왼쪽에서 오른쪽으로 텍스트 표기 기본값
    direction: ltr : 오른쪽에서 왼쪽으로 텍스트 표기
