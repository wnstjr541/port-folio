태그 정리

  • sub : 아래 첨자
  • sup: 위 첨자
  • em : 이탤릭 체(기운모양)로 강조 텍스트
  • i : 이탤릭 체로 표시할 텍스트
  • q : p태그 안에 넣을시 "로 단어를 감싸줌. 짧은 인용문일때 사용
  • blockquote : 긴 인용문일시 사용한다
  • del : 글자 중앙에 실선을 긋는다
  • ins : 글자 밑에 선을 긋는다
  • details : 세부 항목이나 추가 정보를 숨겨 놓았다가 펼쳐 보여줄 수 있는 기능
  • summary : 세부 항목의 제목과 요약 내용
  • dl : 여러 쌍의 정의된 용어 설명을 그룹짓는다.
  • dt : 정의목록에서 용어를 구분한다.(제목)
  • dd : 용어의 정의를 나타낸다.(내용)
    dl의 자식요소는 오직 div, dt,dd 만 가능하다
