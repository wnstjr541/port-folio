파일 및 폴더 네이밍

분류 예시 설명
Folder html 업무별 폴더 생성 00.service, 01.customer, 02.account
html_guid 마크업 가이드, 접근성 가이드, 화면 목록등의 파일 및 폴더 생성 가능
js 라이브러리별 폴더 생성 가능
css 디렉토리별 폴더 생성 가능
img 네이밍별 폴더 생성 가능
HTML 0101.html 화면 코드 사용(별도의 정의된 네이밍 롤이 없는 경우 사용)
00_customer_service_list.html 화면 코드, 메뉴명, 상태의 조합
00_pop_*.html 팝업 파일 사용
00_aj_*.html ajax 이용하여 페이지를 동적으로 사용하는 파일
00_if_*.html iframe 파일 사용
css 00.service.css 폴더명을 사용
default.css 브라우저 호환성을 위하여 기본 css속성을 정의하며 common.css를 @import한다.
layout.css 화면의 레이아웃 구조의 표현을 정의한 파일
common.css 전역에 쓰이는 요소의 표현을 정의하며 html의 업무별, 디렉토리별 구분하여 작성한 css파일을 @import한다.

이미지 예약어

분류 예약어 사용 예 설명
텍스트 h[1~6]_*, h_* h_logo.gif, h2_header.gif heading 요소에 사용
tit_* tit_table_id.gif, tit_definition_list heading 요소에 사용되지 않고 문단의 강조를 나타내는 형식의 텍스트
txt_* txt_msg.gif 문단의 이미지로 된 텍스트 사용
링크요소 nav_* nav_gnb_ko.gif, nav_lnb_about.gif lnb, gnb, snb 등의 네비게이션 요소에 사용
tab_* tab_style_01.gif 네비게이션을 제외한 탭 메뉴의 형태에 사용
btn_* btn_ok.gif 버튼 형식의 a 엘리먼트, button 엘리먼트, input[type=image] 엘리먼트 요소에 사용, 버튼의 이미지가 배경으로 사용하여도 btn_*예약어 사용
박스 box_* btn_ok.gif grouping목적으로 사용되는 div 엘리먼트에 주로 사용
불릿 bul_* bul_arrow.gif 의미가 없는 요소로 사용할수 있는 이미지 사용
아이콘 icon_* icon_search.gif, icon_download.gif 의미 있는 요소로 설명에 대한 텍스트가 명확한 경우
상태 *_on,*_off,*_over, *_out, *_active tab_type_01_on.gif 활성화, 비활성화. 오버/ 포커스 상태에 대한 표현에 사용
예약어 설명
.aside-* 보통 #container 또는 #content에 종속되며 곁가지 영역으로 불리기도 한다.
.gnb 최상위 공통 네비게이션(Global Navigation Bar) 를 지정. 보통 수평 형태로 표현되며 #header에 종속된다. 수직으로 표현되는 경우는 거의 드물지만 그러한 경우 #header에 종속되지 않을수도 있다
.lnb 현재 서비스의 지역(Local Navigation Bar)네비게이션을 지정. 보통 수평 형태로 표현되며 #header에 종속된다. 수직으로 표현되는 경우는 거의 드물지만 그러한 경우 #header에 종속되지 않을수도 있다
.snb .gnb, .lnb를 제외한 측면(Side Navigation Bar)네비게이션 영역을 지정
.nav-* gnb, lnb, snb 이외의 네비게이션을 지정
.section-* 콘텐츠를 분할하거나 그룹핑하는 블럭. 보통 #container 또는 #content 내부에 배치하나 다음 heading 태그 h1~h6와 함께 사용하는 것을 권장한다.
